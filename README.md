# projet-zabbix-monitoring
Projet académique de supervision réseau avec **Zabbix**, réalisé dans le cadre du cursus de **Sécurité Informatique** à l'UIT (Université Internationale de Tunis).
Installation et configuration MySQL/PostgreSQL sur Ubuntu.

##  Description

Ce projet couvre l'installation, la configuration et le déploiement d'une infrastructure de supervision Zabbix sur Ubuntu, avec deux variantes de base de données :
- **MySQL / MariaDB**
- **PostgreSQL**

##  Technologies utilisées

- Zabbix Server 6.0 LTS
- Ubuntu (Linux)
- MySQL / MariaDB
- PostgreSQL
- Apache2 / PHP
-
